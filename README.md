# computer_vision

Algoritmos de Deep Learning para detecção de rostos e objetos

## Instalação do Python 3.7.x

$ sudo apt update
$ sudo apt install software-properties-common
$ sudo add-apt-repository ppa:deadsnakes/ppa
$ sudo apt install python3.7
$ python3.7 --version

Output
Python 3.7.x
